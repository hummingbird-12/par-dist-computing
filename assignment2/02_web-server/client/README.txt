Client
---------------------------

* Instructions
1. Build using the `Makefile`
   $ make

2. Execute the client program
   $ ./client.out SERVER_ADDR SERVER_PORT THREADS_NUM REQ_PER_THREAD REQ_FILES

   Example:
   $ ./client.out 127.0.0.1 9987 5 10 request_files.txt
